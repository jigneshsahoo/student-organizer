Student Organizer
	
A Python-based command-line application for managing student academic activities, including class schedules, task tracking, and note management.

Features

Core Modules
	Schedule Management: Add and view class schedules with timing
	Task Tracking: Manage daily tasks with completion status
	Notes Management: Store and view quick notes and reminders
Technical Features
	Modular architecture with separation of concerns
	File-based data persistence
	Error handling for user inputs
	Menu-driven user interface
Technology Stack
	Language: Python 3.x
	Storage: Text file-based persistence
	Architecture: Modular design pattern
	Version Control: Git & GitHub
Run the application
	python main.py
